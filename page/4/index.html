Keyboard Shortcuts

Sublime：

⌘（command）+]/[ 智能行缩进
⌘（command）+D 选词（反复按快捷键，即可继续向下同时选中下一个相同的文本进行同时编辑）
⌘（command）+P 搜索项目中的文件
⌘（command）+ CTRL + p 打开项目列表面板
⌘（command）+X 删除当前行
⌘（command) +数字 窗口切换
⌘（command) +R 前往 method的函数
⌘（command) +N 新建窗口
Ctrl + Tab 或 Ctrl + PgDn 跳转到下一个打开的标签页
Ctrl + Shift + Tab 或 Ctrl + PgUp 跳转到上一个打开的标签页
