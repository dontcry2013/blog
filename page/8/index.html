Java集合类型

Java 集合可分为 Set、List 和 Map 三种体系：
Set：无序、不可重复的集合。
List：有序，可重复的集合。
Map：具有映射关系的集合。
Collection 接口是 List、Set 和 Queue 接口的父接口，Set 判断两个对象是否相同不是使用 == 运算符，而是根据 equals 方法。
